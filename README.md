# Web端简易图片比例裁剪工具

纯前端、单文件运行的图片裁剪分片工具：拖拽/点击加载图片，默认整张图切成 6×4 网格，可自定义比例与网格，导出 PNG 分片并自动打包 ZIP。

## 使用方法
1. 双击或在浏览器中打开 `index.html`（离线可用，需能访问外部 CDN 加载 Cropper.js 和 JSZip，如需完全离线可将依赖改成本地文件）。
2. 将图片拖到左侧区域或点击选择文件。
3. 右侧可选择比例（默认自由比例，初始覆盖全图），也可调整列数/行数（默认 6×4）。网格覆盖会实时预览。
4. 可拖动/缩放裁剪框，支持滚轮缩放、双击放大、镜像翻转、重新适配。
5. 点击“导出裁剪结果为 ZIP”，自动生成每个网格的 PNG 并下载 ZIP。

在线体验地址：https://image.huangsn.dev/

## 功能点
- 拖拽/点击上传，自动适配整图，实时网格覆盖预览。
- 比例预设：16:9、4:3、1:1、自由比例。
- 网格切分：默认 6×4，可自定义列/行数。
- 导出：高清 PNG 分片 + ZIP 打包下载。
