# image-tool

## 项目介绍
    因为沉迷使用 gemini 生成表情包，一下子找不到有什么网站可以一键切割表情包，于是就有了这个项目。可以到 image.huangsn.dev 使用（用的 cloudflare 的 pages 部署，可能需要魔法）

## 使用方法
1. 双击或在浏览器中打开 `index.html` 选择工具（离线可用；裁剪分片需能访问 CDN 加载 Cropper.js/JSZip，可按需改成本地文件）。
2. 或直接打开 `slicer.html`（裁剪分片）/`scaler.html`（等比例缩放）。
3. **图片裁剪分片**：拖拽/点击/粘贴图片 → 选择比例与网格 → 导出 ZIP（自动生成高清 PNG 分片）。
4. **图片等比例缩放**：输入目标最长边像素 → 拖拽/选择/粘贴图片 → 逐张下载/复制，或点击“下载 ZIP（全部）”批量导出。
5. 各工具的“清空”按钮可重置当前结果，互不影响。

## 功能点
- 单文件纯前端，图片处理全在本地完成。
- 图片裁剪分片：比例预设（16:9/4:3/1:1/自由）、网格自定义、滚轮缩放/双击放大/镜像翻转、导出高清 PNG 分片并 ZIP 打包。
- 图片等比例缩放：批量拖拽/粘贴，最长边缩放保持比例，自动重命名（如 `foo_240x160.png`），逐张下载或复制到剪贴板，内置 ZIP 打包。
